Introduction Quotr in a nutshell

Keywords

Introduction Python Quote management Clean Minimal

Quotr is a Quote Management System. It is used primarily for managing IRC conversations. You can check the example of a QMS at bash.org. Quotr is user friendly. Clean, minimal and easy to use. Since it is made in python it is as secure as the server you're running it on. Quotr works only on Linux for now. It will be ported to Windows as soon as Python for Windows supports certain modules or equivalent libraries are found. Quotr aims to keep things simple, before and after setup. Especially during setup.

Requirements what you'll need

Keywords

Requirements Linux Python2.6 Flask Flask-SQLAlchemy Flask-Bcrypt Flask-WTF SQLite3 Tutorials

You'll require any Linux System with Python 2.6 running to get Quotr setup. If you have a preinstalled version of Python say anything other than 2.6 you can install it in a localized location and trigger all Quotr functions using the new version of Python. In order to do that check out localized python install

Apart from basic Python you'll require:

  1. Flask
  2. Flask-SQLAlchemy
  3. Flask-Bcrypt
  4. Flask-WTF
  5. Py-SQLite3
Please install the latest versions. Here's a short tutorial on working with pip to install these modules. If you require there is a complete dependency setup script which can be downloaded from here.

Setup installing Quotr

Keywords

Setup Command line Database

Quick installation of Quotr wiki requires some commandline work. But don't worry. It's as easy as it gets.

  1. Create a file in the conf folder. Name it onetimeconf.py. It should have the following in it:
    2.                   admin_email = '<your email address>'
    
                  admin_nick = '<your intended nick>'
    
                  admin_pass = '<your intended password>'
  2. Execute the following command
    3. python websetup.py
  3. Edit the appconf.py in the conf folder to edit the name of the database. Read the comments carefully
  4. Follow the instructions and the database will be created first. After that the admin will be setup and the file will be deleted.
  5. This step applies only if the file wasn't deleted. Go to the conf folder and delete the file onetimeconf.py
  6. That's it! You're good to go.

Security issues what can go wrong

Keywords

Setup Command line Debug onetimeconf.py

Quotr was built using Python and Flask, so it is very secure. However if at all you leave the DEBUG key ON for developmental purposes, it can lead to a lot of problems. Always rememeber if you're not developing on Quotr please leave the DEBUG to OFF. Another security issue is typing the password on the command line prompt. Hence the workaround to create the first admin. Kindly delete the file onetimeconf.py if not done automatically once the admin is setup. It is important security issue.

The DEBUG key can be issued a False value manually as well. Go inside the conf folder and edit the file appconf.py
