Introduction Quotr in a nutshell

Keywords
  • Quote Management
  • Python
  • Flask
  • Features
  • Linux

Quotr can be used to manage all types of quotes. It was built to manage IRC conversations. A good example of quote management system is bash.org. Quotr is built on Python and uses the Flask microframework. It was developed on a Linux system and therefore works best with it. Quotr gives the manager the ability to add or remove users.
