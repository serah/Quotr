Requirements what you'll need

Keywords

Requirements Linux Python2.6 Flask Flask-SQLAlchemy Flask-Bcrypt Flask-WTF SQLite3 Tutorials Argparser

You'll require any Linux System with Python 2.6 running to get Quotr setup. If you have a preinstalled version of Python say anything other than 2.6 you can install it in a localized location and trigger all Quotr functions using the new version of Python. In order to do that check out localized python install

Apart from basic Python you'll require:

  1. Flask
  2. Flask-SQLAlchemy
  3. Flask-Bcrypt
  4. Flask-WTF
  5. Py-SQLite3
  6. Argparser
Please install the latest versions. Here's a short tutorial on working with pip to install these modules. If you require there is a complete dependency setup script which can be downloaded from here.
